# NexGenSpec

**On-the-go inspection reporting software.** Denver, CO.

---

## Getting Started

1. **Close Xcode** and delete the old InspectIQ project folder if desired.
2. **Open** `NexGenSpec.xcodeproj` in Xcode.
3. **Select** your development team in Signing & Capabilities.
4. **Build and run** (⌘R) on simulator or device.

---

## Project Structure

- **NexGenSpec/** – All source files (Swift, Assets, Templates)
- Uses `PBXFileSystemSynchronizedRootGroup` – Xcode auto-discovers files in the NexGenSpec folder

---

## Requirements

- iOS 17.0+
- Xcode 15.0+
- Swift 5.0

---

## Clean Slate

This is a fresh project with no InspectIQ references. All branding, bundle ID (`com.nexgenspec.app`), and naming use NexGenSpec.
